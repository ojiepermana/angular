## Design Context

### Users

Pengguna utama adalah tim pengembang aplikasi konsumen yang memakai library ini untuk membangun antarmuka produk mereka.
Konteks penggunaan:

- Menggunakan library dan demo page sebagai referensi implementasi.
- Memakai komponen layout, navigasi, serta override Angular Material dan CDK.
- Membutuhkan styling berbasis Tailwind CSS, dengan prioritas penggunaan class melalui `@apply` untuk override Material.

Job to be done:

- Menyediakan fondasi UI yang konsisten untuk semua aplikasi consumer.

### Brand Personality

Kepribadian brand: dinamis, cool, future.
Emosi utama yang harus terasa: percaya diri.

### Aesthetic Direction

Arah visual: brutal minimal.
Tema: mendukung light dan dark.
Warna acuan utama: rgb(1 53 144).
Referensi: https://www.theverge.com/ (fokus pada kesan editorial tegas, modern, dan percaya diri).
Anti-reference: hindari karakter visual yang terlalu rounded.

### Design Principles

1. Konsistensi lintas aplikasi adalah prioritas utama: setiap komponen harus reusable, predictable, dan mudah diadopsi.
2. Override Material wajib berbasis Tailwind `@apply`; gunakan CSS vanilla hanya jika `@apply` benar-benar tidak memungkinkan.
3. Brutal minimal execution: tegas, minim ornamen, kontras jelas, hirarki informasi kuat, tanpa gaya dekoratif berlebihan.
4. Dual-theme parity: kualitas visual dan aksesibilitas harus setara di light dan dark mode.
5. Confidence-first UX: keputusan tipografi, warna, dan spacing harus menghasilkan kesan profesional, kuat, dan future-facing.
6. Accessibility baseline: harus lolos AXE checks dan memenuhi WCAG AA minimum pada fokus, kontras, dan semantics.
